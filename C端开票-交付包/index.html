📐 使用说明

评审使用

  • 打开 index.html（本文件），点击各卡片快速跳转
  • 或直接双击「点餐-C端开票-PRD.html」阅读需求文档
  • PRD 默认 预览模式（只读），无编辑工具栏，适合评审阅读

开发使用

  • 每个原型文件均为独立 HTML，无需任何依赖，双击即可打开
  • 原型中的表格字段名与 PRD 字段明细完全一致，开发可直接参考
  • 原型中的弹窗、筛选、Tab切换等交互均已实现，可供前后端联调参考

PRD 预览与编辑

  • 交付包内 PRD 为 只读预览模式，无编辑工具栏，适合评审阅读
  • ✏️ 如需编辑：使用项目源文件 点餐-C端开票-PRD.html，点击右下角 ✏️ 按钮进入编辑模式
  • 如果页面内容显示异常，点击「🗑️ 清除缓存」刷新
